Exibir/ocultar arquivos ocultos no Mac (OSX)

Sun 09 February 2014
by rafaelbarrelo
in Mac

tagged finder, mac, dica, osx

No terminal use os seguintes comandos:

  • Exibir:
$ defaults write com.apple.finder AppleShowAllFiles -bool true
$ KillAll Finder
 ```
* Ocultar:
```sh
$ defaults write com.apple.finder AppleShowAllFiles -bool false
$ KillAll Finder

Simples!
